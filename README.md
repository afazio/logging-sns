# Logging::Appenders::SNS

AWS Simple Notification Service (SNS) appender for Logging.

## Installation

Add this line to your application's Gemfile:

    gem 'logging-sns'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install logging-sns
